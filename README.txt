Este aplicativo é um prático sistema de cadastro de pizzaria, construído de forma eficiente com as tecnologias Spring Boot, TypeScript, 
React, Node e Axios. A integração dessas ferramentas proporciona uma solução ágil e moderna para gerenciar informações de uma pizzaria.
Com o Spring Boot no backend, TypeScript e React no frontend, e o Node facilitando a comunicação entre eles, o aplicativo oferece uma 
experiência intuitiva para cadastrar pizzas, clientes e manter um cardápio atualizado. O uso do Axios simplifica as chamadas de API, 
garantindo uma interação suave entre as partes do sistema. Este aplicativo é uma solução simples e eficaz para quem busca uma ferramenta 
ágil e intuitiva para o dia a dia de uma pizzaria.
